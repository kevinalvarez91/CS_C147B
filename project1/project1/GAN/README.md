# GAN Project

## Requirements

- Python 3.10+
- PyTorch 2.0+
- torchvision 0.15+
- numpy
- matplotlib
- jupyterlab

## Setup

Pick any method you prefer.

### uv
```bash
uv sync
```

### conda
```bash
conda create -n gan python=3.11
conda activate gan
conda install pytorch torchvision -c pytorch
pip install matplotlib jupyterlab
```

### venv
```bash
python3 -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install .
```
